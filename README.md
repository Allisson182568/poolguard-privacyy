# POLÍTICA DE PRIVACIDADE — Guardião da Piscina
Última atualização: 2025

O aplicativo Guardião da Piscina (“Aplicativo”, “nós”, “nosso”) tem o compromisso de proteger a privacidade dos usuários. 
Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações ao utilizar o aplicativo.

Ao usar o Guardião da Piscina, você concorda com os termos descritos abaixo.

----------------------------------------------------
1. INFORMAÇÕES COLETADAS
----------------------------------------------------

O Guardião da Piscina coleta apenas as informações necessárias para o funcionamento do aplicativo. 
A coleta pode ocorrer de forma manual pelo usuário ou automaticamente (com permissão).

1.1 Informações fornecidas pelo usuário:
- Medições da piscina (pH, cloro livre, alcalinidade, dureza, temperatura, nível da água, claridade).
- Observações opcionais.
- Configurações da piscina (volume, tipo, profundidade, filtro, moeda usada).
- Fotos da água fornecidas voluntariamente pelo usuário.

1.2 Informações coletadas automaticamente (opcional):
- Localização aproximada (para exibir o clima).
- Permissão de notificações.
- Integração opcional com o Apple Health.

O aplicativo NÃO coleta:
- Nome completo, telefone, endereço, e-mail, CPF, dados financeiros ou informações sensíveis.

----------------------------------------------------
2. USO DAS INFORMAÇÕES
----------------------------------------------------

As informações coletadas são utilizadas exclusivamente para:

- Calcular produtos químicos necessários para tratamento da piscina.
- Registrar histórico e evolução da qualidade da água.
- Gerar relatórios PDF ou backup voluntário.
- Exibir clima local para orientar manutenção.
- Aprimorar recomendações de tratamento.
- Enviar lembretes e alertas de manutenção.

Nenhum dado é enviado para servidores externos. 
Todo o processamento é feito no dispositivo do usuário.

----------------------------------------------------
3. ARMAZENAMENTO E SEGURANÇA
----------------------------------------------------

Os dados são armazenados de duas formas:

3.1 Localmente no dispositivo  
Utilizamos Core Data, que é protegido pela segurança nativa do iOS.

3.2 Backup opcional no iCloud Drive  
Apenas arquivos de backup exportados manualmente pelo usuário são armazenados no iCloud.  
Fotos não são enviadas ao iCloud.  
A equipe do aplicativo não tem acesso a esses dados.

----------------------------------------------------
4. COMPARTILHAMENTO DE INFORMAÇÕES
----------------------------------------------------

O Guardião da Piscina NÃO compartilha dados com terceiros.  
Nenhum dado é vendido, alugado ou enviado para qualquer empresa.  

O usuário pode escolher exportar relatórios ou informações manualmente.

----------------------------------------------------
5. PERMISSÕES DO APLICATIVO
----------------------------------------------------

O app pode solicitar:

- Acesso às fotos → somente para análise opcional da aparência da água.
- Localização aproximada → para exibir clima.
- Notificações → para lembretes de manutenção.
- HealthKit → registrar esforço físico na limpeza da piscina.

O app funciona mesmo se o usuário não conceder as permissões (exceto recursos dependentes delas).

----------------------------------------------------
6. DIREITOS DO USUÁRIO
----------------------------------------------------

O usuário pode:

- Excluir medições individualmente ou todo o histórico.
- Exportar seus dados.
- Revogar permissões a qualquer momento.
- Reinstalar o app para limpar todos os dados locais.

Não armazenamos dados após exclusão.

----------------------------------------------------
7. PRIVACIDADE DE CRIANÇAS
----------------------------------------------------

O aplicativo não é direcionado a crianças e não coleta intencionalmente informações pessoais de menores.

----------------------------------------------------
8. ALTERAÇÕES NA POLÍTICA
----------------------------------------------------

Podemos atualizar esta Política de Privacidade periodicamente.  
A data de atualização sempre será informada no início do documento.
